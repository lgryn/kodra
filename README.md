# Kodra

AI-powered CLI for modern development workflows

## Features

- explain a code file
- built with Node.js + TypeScript
- uses OpenAI API

## Setup

```bash
npm install
```

Create `.env`:

```env
OPENAI_API_KEY=your_key
```

Run:

```bash
npm run dev -- explain src/cli/index.ts
```

## Stack

- Node.js
- TypeScript
- OpenAI API
- Commander
- Zod
